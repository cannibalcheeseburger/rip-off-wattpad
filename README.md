## Installation

 - [Python 3.7.6](https://www.python.org/downloads/)


 - Build from source

```
git clone https://github.com/cannibalcheeseburger/cpu-scheduling-simluation.git
cd cpu-scheduling-simulation
```


 - Requirements
```
python -m pip install -r requirements.txt
```

 - Running program
 ```
python book.py
 ```
